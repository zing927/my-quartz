# 欢迎来到我的博客 🎉

这是我使用 Obsidian + Quartz 发布的第一篇文章！

## 关于这个博客

这个博客使用以下技术栈搭建：

- **[Obsidian](https://obsidian.md)** - 强大的笔记应用
- **[Quartz](https://quartz.jzhao.xyz)** - 静态站点生成器
- **GitHub Pages** - 免费网站托管

## 功能特点

### ✨ 写作体验

- 在 Obsidian 中像平时一样写笔记
- 使用 Markdown 语法
- 支持双向链接 `[[其他笔记]]`
- 支持标签 `#标签`
- 支持代码块、数学公式、图表

### 🚀 自动化发布

- Quartz Syncer 插件自动同步
- Quartz 自动生成静态网站
- GitHub Actions 自动部署

### 📱 响应式设计

- 完美适配手机、平板、电脑
- 深色模式支持
- 搜索功能
- 图谱视图
