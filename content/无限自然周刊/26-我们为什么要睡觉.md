# 02.张云帆聊《芒格之道》

**1.搞明白基本的底层逻辑**

例如“多巴胺”让我们快乐，“内啡肽”也会让我们快乐。那么我们要搞明白两则之间的差异，真正搞懂，而且我们还要利用它改变我们的行为。这样才算真正懂了一个道理。然而大部分人，就满足于我知道了的，并不会去践行。导致我们稀里糊涂的不知道什么原因失败？也不知道为什么会成功，都是凭感觉。

将道理内化，吃透，不实践，慢慢建立自己的方法论和原则。去实践，就可以拉开大部分人的差距。因为大部分人止于“我知道”。

**2.时间管理**

时间管理，不只是定好时间在什么时间做什么事。而是把自己放到一个正向循环的系统中。举个例子：你想早起，可是你并没有早睡，也没有规律的饮食习惯，也不自信自己能不能做成这件事，或则坚持多久等等等等都会导致你无法早起。

1.睡眠，是让自己走向正向循环的基础。我们为什么要睡眠，这本书提到睡眠可以让我们更聪明。因为学习到的东西的记忆是短暂的，睡眠可以让这样信息更深的记忆。也可以调节人的情绪，间接影响一个人的情商。也可以明显提升第二天的做事，思考效率。

2.把24小时分时间块。例如睡眠时间，工作时间，陪家人孩子，通勤时间等。让这些时间成为固定。这样可以让我们个人这个系统更稳定，而不是混乱的状态。混乱，是我们长期以来没有突破，没有收获的原罪。除去那些固定的时间，还有剩余的个人时间。这块时间需要我们想明白自己想要什么，做成为什么样的人，相信自己做成什么事。这样我们才会更好的安排这块时间。

- 保持大脑开放的状态
- 不断获取新的信息，刷新认知
- 阅读运动，做自己真正喜欢的事

# 04.复杂表单如何设计优化

![](https://cdn.nlark.com/yuque/0/2023/png/728168/1700976939716-a1d9ffa0-a7c3-495d-96dd-ba016cd73bd3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_31%2Ctext_TGlua-iuvuiuoeWRqOWIig%3D%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

复杂表单的设计优化需要考虑以下几个方面：

1. 页面布局优化：采用左右结构的版式，将核心内容放在左侧，次要内容放在右侧，增加页面的可视范围和填写效率。
2. 信息预览功能：在页面上增加一个预览按钮，点击后可以实时预览填写的内容，并能够快速定位到相应字段位置，方便用户确认和调整。
3. 错误提示优化：将错误信息集中展示在一个区域，让用户快速了解错误内容，并提供点击跳转到具体错误位置的功能，方便用户进行修改。
4. 善用重复：针对重复的字段，可以提供模版和复制的功能，让用户可以快速填写相同或类似的内容，提高表单使用效率。

# 09.REM睡眠

检测自己的睡眠质量已经有段时间了，发现睡眠时间差不多在7-8小时左右，rem值18%以上，第二天的精力就还不错。

《为什么我们要睡觉》中提到REM睡眠的重要性：

- 巩固记忆：REM睡眠阶段，会把白天获取的各种信息与大脑中相关的内容进行关联，并形成新的记忆；
- 促进大脑发育：新生儿常常以REM睡眠为起始阶段，且50%以上的睡眠时间都处于REM睡眠中；
- 减压等情绪改善：REM睡眠可以帮助人从各种压力中恢复；特别是在创伤事件后，更多的REM睡眠会减少患上PSTD（创伤后应激障碍）的概率；
- 改善精力、提高注意力等
- 完整充足的REM有利于控制体重

如何获得更多的REM睡眠？

- 规律睡眠：每天在同一时间入睡和醒来，即使在节假日也是如此；
- 及时发现并治疗睡眠障碍：如经过有效治疗的阻塞性睡眠呼吸暂停，REM片段化明显减少，REM占睡眠时长比例增加；
- 避免服用抑制REM睡眠的药物：如某些抗抑郁类药物；可以咨询你的医生或睡眠专科医生选择合适的药物；
- 避免酒精、咖啡因、烟草：这些都会导致REM睡眠延迟或减少；应尽量避免使用，特别是在下午和晚上；
- 保持良好的睡眠卫生习惯：良好的睡眠卫生习惯是确保REM睡眠的重要前提
- 定期锻炼
- 保持良好、安静、黑暗的睡眠环境
- 睡前避免剧烈活动，多做利用身心放松的事情
- 睡前1小时不要看手机、电视、电脑等发光的屏幕
- 卧室中避免有任何提示时间的工具；
- 如果躺在床上20min仍然没有任何睡意，那就要起床，离开卧室，去其它房间，直到感觉困倦时才回到卧室上床睡觉。
