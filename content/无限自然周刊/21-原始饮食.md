# 05.原始饮食？

因为《真原医》这本书，了解到易楚，是一位倡导原始饮食的健康专家，他推崇的原始饮食主要包括以下几个方面，和生酮饮食、《真原医》里面的饮食结构都有些差异。

原始饮食：

1. 食用大量蔬菜和水果，特别是深色蔬菜和浆果类水果，因为它们富含抗氧化剂和营养素。

2. 食用野生动物肉类，比如鱼、鸟类、兔子等，因为它们富含高质量的蛋白质和脂肪，并且没有工业化养殖带来的潜在危害。

3. 避免或减少加工食品、精制糖和高盐、高脂肪的食物的摄入，因为这些食物可能会对身体健康造成负面影响。

4. 食用天然的油脂，比如橄榄油、椰子油、牛油等，而避免使用工业化加工的油脂。

5. 注重饮食的多样性和平衡，确保摄入足够的营养素和能量。

地址：[https://www.zhihu.com/column/paleo](https://www.zhihu.com/column/paleo)

原始饮食和生酮饮食有一些相似之处，比如都强调食用天然、健康的食物，并且避免加工食品和精制糖的摄入。但是，它们也有一些明显的不同之处：

1. 原始饮食更加注重蛋白质和脂肪的摄入，而生酮饮食则更加注重脂肪的摄入，同时限制碳水化合物的摄入。
2. 原始饮食可以包括一些天然的碳水化合物，比如水果和根茎类蔬菜，而生酮饮食则会限制这些食物的摄入。
3. 生酮饮食可能需要补充一些营养素和维生素，因为限制碳水化合物的摄入可能会导致某些营养素不足。
4. 生酮饮食通常需要严格控制每日的摄入量和比例，以达到特定的代谢状态，而原始饮食则更加注重饮食的多样性和平衡。

  

# 06.目标与系统

斯科特·亚当斯曾经提出过一个有趣的观点，即“目标是无用的”。他认为，设定目标往往会让人们过于专注于未来的结果，而忽视了现在正在发生的事情。相反，他建议人们应该专注于建立系统，即一系列能够帮助人们达成目标的行动和习惯。通过建立系统，人们可以更好地管理自己的时间和资源，更容易实现他们想要的结果。

假如设定系统，认真完成作业，建立错题库，课前预习，课后复习，大量阅读……长期在这样的良性系统中，一个人很难不优秀。

系统不关注结果，只关注过程。只要过程是正确的，未来就有无数种可能。

# 08.提起设计系统，别再只会说组件

在旁听一些分享、复盘、汇报时发现很多设计师总把设计系统挂在嘴边，但谈来谈去都还是“组件”那套，什么设计模式、原子设计、design token讲来讲去最后都是在讲“组件”。设计系统或者设计语言就像人类的自然语言，是一个由字、词、句、语法组成的沟通系统，只有组件就好像只有字词，无法达到理想的沟通效果。这也是为什么我们总说设计系统（或组件）是为了提高生产效率、降低沟通成本，而实际上使用时却常常成为设计绊脚石的原因之一。
